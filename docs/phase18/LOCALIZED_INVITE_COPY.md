## 4. Privacy Reminder

The `invite.privacy_reminder` key warns users about sharing women-only / verified-only invite links publicly:

> Anyone with this link can preview and join WalkTogether. Don't share publicly if your walk is women-only or verified-only — share directly with trusted contacts instead.

This is translated across all 9 languages.

## 5. How Localization Works

The `ShareSheet` component uses the `getShareCopy()` function to retrieve the localized default message:

```typescript
function getShareCopy(lang: LanguageCode, targetType: TargetType): string {
  const keyMap: Record<TargetType, string> = {
    app: "invite.default_app_message",
    area: "invite.default_area_message",
    group_walk: "invite.default_group_walk_message",
    club: "invite.default_club_message",
  };
  const fallback: Record<TargetType, string> = { /* English fallbacks */ };
  const translated = translate(lang, keyMap[targetType]);
  return translated === keyMap[targetType] ? fallback[targetType] : translated;
}
```

If the translation is missing, the English fallback is used.

---

## 6. User-Editable Message

The ShareSheet allows users to edit the default message before sharing. The localized default appears in the textarea, and the user can customize it. The edited message is passed to the invite link creation API and stored as the link's message.
