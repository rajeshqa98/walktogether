# WalkTogether — Community Host Program

**Free. No monetization. Community-driven.**

---

## 1. What Is a Community Host?

A Community Host is a verified WalkTogether user who takes on a leadership role in their local walking community. Hosts create group walks, start walking clubs, welcome new users, and help maintain a safe, friendly walking environment.

### What Hosts Can Do
- Create group walks at public meeting points
- Start and manage walking clubs
- Send announcements in group chat
- Welcome new walkers in their area
- Organize women-only or senior-friendly walks
- Report unsafe behavior to admin

### What Hosts Cannot Do
- See exact private locations of participants
- Force users to share live location
- Message users outside the group walk context
- Charge money or collect fees
- Restrict who can join public group walks

## 2. Host Requirements

| Requirement | Criteria |
|-------------|----------|
| Verified profile | Selfie verification completed |
| Trust score | ≥60 (out of 100) |
| Account standing | No unresolved safety reports |
| Completed walks | ≥3 completed walks (demonstrates engagement) |
| Account age | ≥7 days since signup |
| Safety guidelines | Must read and agree to host safety guidelines |

---

## 3. Host Safety Guidelines

1. **Public meeting points only** — Always choose parks, cafes, landmarks, or other public spaces. Never private residences.
2. **Set reasonable group size** — For your first walk, keep it to 8-10 participants. You can increase as you gain experience.
3. **Send a pre-walk announcement** — Use the group chat announcement feature to share meeting point, time, and what to bring.
4. **Welcome new walkers** — Be inclusive. Greet everyone. Help nervous first-timers feel comfortable.
5. **Do not share participants' private information** — No phone numbers, addresses, or personal details.
6. **Cancel if unsafe** — If weather is bad, the meeting point feels unsafe, or you have concerns, cancel the walk. Safety first.
7. **Report unsafe behavior** — If any participant behaves inappropriately, use the report function and contact admin.
8. **You cannot force location sharing** — Participants choose whether to activate safety share. Respect their choice.
9. **End the walk properly** — Use the "End walk" button when the walk is complete. This triggers ratings and properly closes the session.
10. **Be a role model** — Your behavior sets the tone for your walking community. Be kind, respectful, and safety-conscious.

## 4. Host Recognition

Hosts are recognized in the app:
- **Trust score boost:** +5 when first group walk is completed successfully
- **Host badge:** Visible on profile (future v1.2 feature)
- **Priority support:** Host safety concerns get 2-hour SLA
- **Community highlight:** Hosts featured in area dashboard

---

## 5. Host Recruitment Process

### Automated Identification
The system identifies potential hosts based on:
- Verified profile + trust score ≥60
- 3+ completed walks
- Located in an area with >5 users but 0 group walks

### Recruitment Flow
1. Admin sends invitation: "You're invited to become a WalkTogether Community Host!"
2. User reads host safety guidelines
3. User agrees to guidelines
4. User creates first group walk
5. Admin monitors first walk for safety
6. On successful completion: trust score +5, host status confirmed

### Self-Nomination
Users can also self-nominate:
- Settings → "Become a Community Host"
- Must meet all requirements
- Reads and agrees to guidelines
- Creates first group walk

---

## 7. Host Removal

A host can lose host status if:
- Trust score drops below 50
- Receives 2+ unresolved safety reports
- Violates host safety guidelines
- Account is suspended or banned

Host removal is documented in the audit log. The user can re-qualify after improving trust score and resolving reports.
