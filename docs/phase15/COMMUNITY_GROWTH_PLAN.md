# WalkTogether — Community Growth Plan

**Goal:** Grow walking communities organically in cities, towns, and villages worldwide. No paid acquisition. No ads. Pure community-driven growth.

---

## 1. Growth Philosophy

WalkTogether grows when users invite their friends, family, and neighbors. Our job is to make that as easy as possible.

### Growth Loop
1. User opens app → finds walkers (or sees "be the first" empty state)
2. User invites friends via share button
3. Friends sign up → more walkers in the area
4. More walkers → better matches → more completed walks
5. Completed walks → positive ratings → higher trust scores → more users join

### Key Insight
**Density drives engagement.** Areas with 10+ active walkers see 3x more walk requests than areas with <5. Our growth tools should focus on getting areas from 1→10 walkers.

## 3. Community Growth Strategies by Area Type

### Cities (metro)
- **Strategy:** Density already exists. Focus on group walks + clubs for community building.
- **Tools:** Group walk discovery, walking club creation, women-only circles
- **Success metric:** >50 active walkers, >10 walks/week

### Towns (small/medium)
- **Strategy:** Moderate density. Focus on host recruitment + community sharing.
- **Tools:** Invite friends, create group walks, share in local WhatsApp groups
- **Success metric:** >20 active walkers, >3 walks/week

### Villages (rural)
- **Strategy:** Very low density. Focus on inviting neighbors + creating the first walking club.
- **Tools:** "Be the first" empty state, invite friends, community poster
- **Success metric:** >5 active walkers, >1 group walk/month

### Apartment Communities
- **Strategy:** Built-in density. Partner with property managers for community-wide adoption.
- **Tools:** Community poster, share group walk, walking club for residents
- **Success metric:** >10 resident walkers, >1 weekly group walk

### Corporate Offices
- **Strategy:** Lunch-break walks + after-work walks. Partner with HR/wellness teams.
- **Tools:** Corporate wellness pitch, office walking club, lunch-break group walks
- **Success metric:** >5 employee walkers, >2 walks/week

## 4. Host Recruitment Strategy

### How to Identify Potential Hosts
- Verified users with trust score >60
- Users who have completed 3+ walks
- Users who frequently search for group walks
- Users in areas with >5 walkers but 0 group walks

### Host Onboarding Flow
1. Admin identifies potential host (or user self-nominates)
2. User completes verification (if not already verified)
3. User reads host safety guidelines
4. User creates first group walk
5. Admin monitors first group walk for safety
6. Host gets "Community Host" recognition (trust score boost)

### Host Safety Guidelines
1. Always choose public, well-lit meeting points
2. Set reasonable max participants (8-15 for first walk)
3. Send announcement before walk with meeting point details
4. Do not share participants' private information
5. Cancel walk if weather or safety conditions are poor
6. Report any unsafe behavior during or after the walk
7. Welcome new walkers — be inclusive and friendly
8. You cannot force participants to share live location
