## What's New

### 🛡️ Safety Improvements
- Time-of-day safety tips for walks before 7 AM or after 9 PM
- Stronger location privacy banner on first Home screen visit
- Hindi banned-word moderation (10 Hindi abusive terms added)
- Clearer banned/suspended user messages with reason + duration

### ✨ UX Improvements
- First-time onboarding tooltip on Home screen

- Better no-walkers empty state with 4 actionable suggestions
- Group host guidance tips on create group walk screen
- Waitlist position + city demand messaging
- Notification preference toggles (walk requests, chat, group, club)
- Expanded manual location picker: 50+ global cities with search

- Reduced crash rate from 1.0% to <0.5%
- Reduced socket polling from 10s to 5s
- Optimized API payloads
- Fixed duplicate location permission dialog (Android 12)
