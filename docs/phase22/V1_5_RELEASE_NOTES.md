# WalkTogether — v1.5 Release Notes (Phase 22 Update)

**Phase:** 22
**Release:** v1.5 (updated)
**Tagline:** Fair, explainable, recoverable — with better admin tools.

---

## What's New in Phase 22

### Admin Workload Management
- Safety task auto-assignment to least-busy admin
- Manual assignment and reassignment
- Due dates for high-priority tasks (4-hour SLA)
- Overdue task tracking
- Admin workload dashboard at `/admin/workload`

### Host Badge UI
- Host badge now visible on group walk cards
- Badge appears in Groups list, Group Walk Detail, Club Detail

### Community Governance Center
- User-facing governance page with 9 sections
- Covers safety, reporting, appeals, host responsibilities, women-only rules, privacy, emergency
- Accessible from Settings → Community Guidelines

### Expanded Moderation
- Kannada: +3 terms
- Bengali: +3 terms
- Spanish: +6 terms
- Arabic: +4 terms
- French: +3 terms

### Governance Metrics
- New API: `GET /api/admin/governance-metrics`
- Tracks appeals, false positives, enforcement, outcomes

### Trust Score Transparency
- Trust score explanation on appeal screen
- Factors explained without exposing formula
- Users can appeal trust score decisions

## Free Product Promise

WalkTogether is and always will be free. No premium, no subscriptions, no ads. Safety features are mandatory and always free.

— The WalkTogether Team
