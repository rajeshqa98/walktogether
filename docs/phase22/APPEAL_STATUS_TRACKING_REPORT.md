# WalkTogether — Appeal Status Tracking Report

**Phase:** 22
**Status:** Enhanced (Phase 21 base + Phase 22 context)

---

## 1. Overview

Users can track their appeal status. Admins see full context.

## 2. User-Facing Status

- Submitted → Under Review → Needs More Info → Approved/Rejected/Closed
- Admin response visible
- Next steps clear

## 3. Admin Context (Phase 22 enhanced)

- Related reports count
- Related safety tasks
- Previous appeals
- Trust score (current)
- User account status
- Admin notes
- Previous admin actions (from audit log)

## 4. Notifications

- Appeal submitted → admin notified
- Under review → user notified
- More info requested → user notified
- Approved → user notified, account/host/trust restored
- Rejected → user notified
