# WalkTogether — Trust Score Transparency Report

**Phase:** 22
**Status:** Implemented

---

## 1. Overview

Trust score is explained to users without exposing the exact formula.

## 2. What Users See

- Trust score number (0-100) on walker card and detail
- High-level factors: verification, completed walks, ratings, safety signals, reviewed reports
- "Reports trigger review, not automatic punishment"
- "False positives can be corrected"
- "You can appeal any restriction"

## 3. What Users Don't See

- Exact scoring formula
- Specific weights for each factor
- Details that could help gaming the score

## 4. Where It Appears

- Appeal screen (trust score transparency section)
- Walker card (numeric score)
- Walker detail sheet (numeric score)

## 5. Principle

Transparent but not formulaic. Fair and recoverable. Reports trigger review, not punishment.
